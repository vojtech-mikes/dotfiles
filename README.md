Obsahuje konfig pro

- zsh
- kitty
- nvim

Taky potřebuješ

- fzf
- fd
- ripgrep
- black
- isort
- stylua
- p10k

Všechny dependence můžeš můžeš nainstalovat pomocí chell scriptu `setup.sh`

```bash
# Add rwx permission to the current user
chmod 700 ./setup.sh
# Launch the script
./setup.sh
```
